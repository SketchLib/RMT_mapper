-----------PYCPX-----------
Use modified version at https://github.com/lavanyaj/pycpx.git

sudo apt-get install gcc
sudo apt-get install gcc-c++
sudo apt-get install python-numpy
sudo apt-get install cython
sudo apt-get install python-dev (fixes Python.h errors)

git clone https://github.com/lavanyaj/pycpx.git
cd pycpx

For my macbook, I had to change 2 lines in one of the .cpp files generated by "python setup.py --cython install --user" so that it would compile. Hopefully you won't need to do that. In case you do, I've also uploaded the .cpp file I changed (pycpx/pycpx/my_pyconcert.cpp)

vim .bashrc, .profile
export CPLEX_PATH=/opt/ibm/ILOG/CPLEX_Studio126

----installation command----
python setup.py --cython install --user
python setup.py install --user
